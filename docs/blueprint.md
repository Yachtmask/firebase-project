# **App Name**: GhostDrop

## Core Features:

- Aptos Wallet Connect & Status: User interface for securely connecting and disconnecting Aptos wallets, and displaying the connected wallet address, fully integrated with the latest Aptos Wallet Adapter for broad compatibility.
- Secure File Upload & Encryption: Functionality allowing users to upload sensitive files, which are then encrypted client-side and securely stored within their GhostDrop vault, preparing them for the dead man's switch mechanism.
- Countdown & Daily Check-in: A clear, real-time countdown timer and an intuitive interface for users to perform regular daily check-ins to reset the timer, preventing the automatic release of their vault contents.
- Recipient Management: Users can securely designate and manage a list of beneficiaries who will automatically gain access to the vault contents if the dead man's switch is triggered.
- Automated Vault Release: Robust backend logic designed to automatically decrypt and securely distribute the stored files to the designated recipients if the check-in countdown expires.
- Shelby Standard Interoperability: Core system implementation ensuring all vault operations, file management, and release mechanisms seamlessly conform to the latest Shelby project APIs and SDKs for reliable and standardized integration.

## Style Guidelines:

- Primary color: A deep, sophisticated purple-blue (#52338F) to evoke reliability and a modern digital feel for key interactive elements.
- Background color: A very dark, subtle grayish-purple (#17161C) to provide a sleek, secure backdrop that emphasizes the digital vault concept.
- Accent color: A vibrant, clear blue (#57B2F2) used sparingly for calls to action, important notifications, and highlighting crucial information.
- Headline and body font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for conveying serious information with clarity.
- Use minimalist, secure-themed icons for actions like lock/unlock, upload, countdown, and wallet connections, maintaining a consistent modern aesthetic.
- Employ a structured and clean layout with ample spacing, emphasizing important data and user actions through clear visual hierarchy appropriate for a security-focused application.
- Incorporate subtle and smooth animations for transitions between states (e.g., wallet connection status, countdown updates) to provide a polished and responsive user experience without being distracting.
